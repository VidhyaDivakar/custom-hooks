# React + TypeScript + Vite

Pagination - Custom Hooks

This project is about building custom React hooks, mainly focusing on pagination logic. It keeps the logic separate from the UI so everything stays clean and reusable. The usePagination hook manages page numbers, navigation, and slicing the data properly. A demo component is used to show how the hook works with a list of items in a practical way. It also includes features like changing items per page, moving between pages, and showing the current item range. Overall, the idea is to build something that behaves like real-world pagination while keeping the structure simple.
